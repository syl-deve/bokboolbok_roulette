# Changelog

## [1.0.3] - 2026-03-17

### 2026-03-17
#### 추가
- **인앱 결제(IAP) 도입** — "광고 제거" PRO 버전 구매 기능 추가
- **PRO 버튼** — 상단 바(AppBar)에 세련된 그라데이션 PRO 구매 버튼 추가
- **광고 제거 로직** — 결제 완료 시 하단 배너 및 전면 광고가 상시 제거됨
- **구매 복원** — 기기 변경 시 기존 구매 내역을 복원하는 기능 포함
- **IAP 전용 서비스** — `in_app_purchase` 라이브러리를 활용한 `IAPService` 구현

#### 수정
- **버전 업데이트** — v1.0.3 (Build 12) 패키징 및 프로덕션 빌드 완료
- **PRO 버튼 UI 개선** — '광고 제거' 문구 추가 및 다국어 지원 (ko, en, ja, zh)
- **구매 복원 기능 가시화** — AppBar 상단에 수동 구매 복원 버튼 추가

## [1.0.2] - 2026-03-06

### 2026-03-06
#### 추가
- **사운드 효과** — 스핀 중 틱틱 루프 효과음, 당첨 시 팡파레 효과음 (`audioplayers`)
- **햅틱 피드백** — 스핀 시작 시 중간 진동, 당첨 확정 시 강한 진동 3회 연속

#### 수정
- 참가자 목록 아래 모바일 불필요한 여백 제거 (`ListView` → `Column` 교체)
- 프리셋/히스토리 버튼 위아래 간격 균일화 (각 12px)

### 2026-03-04
#### 추가
- **프리셋 저장/불러오기** 기능 추가
  - 참가자 목록을 이름을 붙여 프리셋으로 저장
  - 저장된 프리셋 목록에서 원터치로 참가자 목록 교체
  - 프리셋 삭제 (확인 다이얼로그)
  - 앱 재시작 후에도 프리셋 유지 (SharedPreferences 저장)
  - 프리셋 미리보기: 참가자 최대 3명 이름 + 총 인원 수 표시
  - 4개 언어 지원 (ko / en / ja / zh)

#### 수정
- **전면광고 닫힌 후 불필요한 3초 딜레이 제거** — 광고 닫는 즉시 스핀 시작
